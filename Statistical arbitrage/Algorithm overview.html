1.1. How our algorithm works: High Level Overview¶

  1. Identify Pairs (('TOTAL', 'UNILEVER'), ('SAP', 'ASML'), ('ALLIANZ', 'LVMH'))
  2. Set thresholds and assign weight limits to thresholds
  3. For each theshold $i$:
    • If z-value lies between threshold $i$ and the next highest threshold $i+1$, take a position in the pair up to the risk limit
    • If z-value is below the next lowest threshold $i-1$, then get out of positions in line with risk limits
  4. Check hedging:
    • If hedging outside hedging thresholds, try to get out of positions to improve hedging if profitable
    • If not profitable to get out of position, prioritise improving other side of pair (setting signal)
  5. Adaptive Thresholds:
    • Make a small change in threshold
    • Record profit/loss for next $n$ iterations
    • If profit goes up, continue chaging threshold in the same direction (as a function of the increase in profits)
